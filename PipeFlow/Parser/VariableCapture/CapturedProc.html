#decorate {|current_proc, *args, &block| ... } ⇒ CapturedProc

Note:

It is expected that the decorator function call the original implementation at some point.

Create a new proc with some additional operations performed around the current implementation. This works by passing the current Proc and received arguments/block to the decorator.

Yield Parameters:

    The current value of this captured proc

    received block
