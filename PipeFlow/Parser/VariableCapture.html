#bound_local_variables ⇒ Array<CapturedVar>

Collect all captured variables from the local environment. The types representing the variables are determined by the types of the values. That is, if a variable has a value that is also a Proc, then it will be represented as a CapturedProc. The default is CapturedVar.

Returns:

  • (Array<CapturedVar>) —

    all local variables in the bound environment
