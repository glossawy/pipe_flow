Overview

A DSL for defining visit methods for various types, with support for aliasing on types and methods. The main utility is deriving the visit method name that is compatible with Visitor#visit.

def alias_visit(type, to:)
  case to
  when Class then alias_to_type(type, to)
  when String, Symbol then alias_to_method(type, to)
  else raise TypeError, 'Can only alias visit to types (Classes) or ' \
                        'method names (Strings/Symbols).'
  end
end
